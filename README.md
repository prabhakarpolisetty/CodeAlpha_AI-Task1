## Task 1: Language Translation Tool

This project is a web-based Language Translation Tool developed to help users translate text easily between different languages. The application provides a simple and user-friendly interface where users can enter text, choose the source language and target language, and receive the translated output instantly.

The system integrates with a Translation API such as Google Translate API or Microsoft Translator API to process the input text and return accurate translations in real time.

### Features

- User-friendly interface for text input
- Source language selection
- Target language selection
- Real-time translation using Translation API
- Clear display of translated text
- Copy translated text option for easy reuse
- Optional text-to-speech support for better accessibility

### Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Express.js (if applicable)
- API Integration: Google Translate API / Microsoft Translator API
- Deployment: Localhost / Web Hosting Platform

### Working Process

1. User enters the text to be translated.
2. User selects source and target languages.
3. The system sends the request to the Translation API.
4. The API processes the text and returns the translated result.
5. The translated text is displayed clearly on the screen.
6. Users can copy the translated text or use text-to-speech (if enabled).

### Objective

The main objective of this project is to provide fast, accurate, and user-friendly multilingual communication support through a responsive web application.
